Burpsuite安装及配置

发表于2022-10-02|更新于2022-10-02|渗透测试工具+环境抓包工具
|字数总计:1.2k|阅读时长:5分钟|阅读量:|评论数:

Burp Suite是一个集成化的渗透测试工具，它集合了多种渗透测试组件，使我们自动化地或手工地能更好的完成对 Web 应用的渗透测试和攻击。

Burp Suite是由Java语言编写而成，而依托Java自身的跨平台性，使得软件在不同系统平台使用更加方便。

安装

网上很多安装激活的方法，此处不赘述，记录的是关于vbs脚本自动化操作

百度网盘下载：

缩小 jar 包体积新版的 BurpSuite

默认下载的 jar 格式的版本里面包含了四个平台的 chromium包：

只保留Windows平台的 chromium 包:

原先大小为 498M ，删除之后瘦小到了 182M

运行脚本

Windows 平台，鼠标双击运行文件：BurpSuite2022.1.1.vbs

Linux 平台，终端执行命令： sh BurpSuite2022.1.1.sh

burp安装过程中的一些问题：

原先jdk版本太低，导致vbs脚本一起运行不起来，但是不允许也是可以破解burp的，只不过强迫症患者一定要去解决自己电脑上出现的问题。将java version “1.8.0_271”升级为17版本

（老师附带的jre，好像不行，不知道为什么）

emm,升级了jdk为14，17版本，BurpSuiteLoader2022.1.1.jar文件均运行不起来

后来发现 ：是目录有中文的原因。。将目录改成全英文，emm就可以了

安装BurpSuite时，配置完Java环境后burp-loader-keygen.jar无法打开的问题：

  • 新版的burp需要jdk8以上版本才可以运行，在老师给的burp文件夹里面附带了14版本的jre
  • 老师写的vbs脚本可以一键运行burp

https抓包

https证书下载

方法一：

保存证书到 burpsuite.der 文件

方法二：

浏览器设置代理 127.0.0.1:8080 后，访问 http://burp ，点击页面 CA Certificate 按钮，下载证书 cacert.der 。

安装证书

双击打开 burpsuite.der 证书文件，选择安装证书,选择，将所有证书都放入下列存:储：受信任的根证书颁发机构

firefox 浏览器导入证书

点击火狐浏览器右上角栏→选项→右上角搜索处搜“证书”→证书颁发机构->导入->选择下载的 cacert.der 证书->信任证书->确定

查看证书，有了 PortSwigger 机构的证书，导入成功

chrome/edge 浏览器导入证书

输入 chrome://settings/ 进入浏览器设置，在设置中搜索管理证书

安全 > 管理证书 > 受信任的根证书颁发机构 > 导入 > 选择 burpsuite.der

其他问题

火狐浏览器抓包问题

问题：总是有到 http://detectportal.firefox.com/success.txt 的请求流量

解决：

  1. 在浏览器的地址栏输入 about:config 。
  2. 输入 network.captive-portal-service.en ，进行查找。
  3. 本来该项的值是 true ，双击 true 这个地方，改成 false 即可。
  4. 修改完以后，直接关闭，无需保存，就不会再出现上述的数据包了。
