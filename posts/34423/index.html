一些补充点

查看官方文档工具——velocity破解

Velocity - Windows 的文档和文档集查看器 (silverlakesoftware.com)

Velocity是Windows下的Dash替代软件，可以方便的查看相关开发文档。不过该工具需要收费，有能力还是支持正版软件。

反编译破解：

  1. 下载反编译软件 dnSpy，下载32位版本
  2. 打开dnSpy.exe，并使用dnSpy打开Velocity.exe
  3. 找到对应的OnShown函数，右键后点击编辑方法，需要将FromSenconds(8.0)改为FromSenconds(0.0)。

具体如下：

改为IL的方式打开代码（默认是C#）

右击编辑IL指令：

将8改为0

编译保存即可

导包

  • from xxx import xxx是精准导包，从xxx包带入xxx类
  • import xxx 直接导入整个文件
  • 一般建议精准导入

文件执行入口

main下面的函数属于本文件，如果本文件被导入其他文件执行，在文件执行入口main下面的部分不会被执行
