Docker - 常见命令

|字数总计:974|阅读时长:3分钟|阅读量:|评论数:

Docker 备忘单

原文：https://github.com/xcad2k/cheat-sheets/blob/main/infrastructure/docker/docker-cli.md

容器管理

命令 描述
docker create IMAGE 创建一个新容器
docker start CONTAINER 启动一个容器
docker stop CONTAINER 优雅地停止容器
docker kill CONTAINER 杀死（SIGKILL）一个容器
docker restart CONTAINER 优雅地停止和重启容器
docker pause CONTAINER 挂起一个容器
docker unpause CONTAINER 恢复一个容器
docker rm CONTAINER 销毁一个容器

容器批量管理

命令 描述
docker stop $(docker ps -q) 停止所有正在运行的容器
docker stop $(docker ps -a -q) 停止所有已停止和正在运行的容器
docker kill $(docker ps -q) 杀死所有正在运行的容器
docker kill $(docker ps -a -q) 杀死所有停止和运行的容器
docker restart $(docker ps -q) 重新启动所有正在运行的容器
docker restart $(docker ps -a -q) 重新启动所有已停止和正在运行的容器
docker rm $(docker ps -q) 销毁所有正在运行的容器
docker rm $(docker ps -a -q) 销毁所有停止和运行的容器
docker pause $(docker ps -q) 暂停所有正在运行的容器
docker pause $(docker ps -a -q) 暂停所有已停止和正在运行的容器
docker start $(docker ps -q) 启动所有正在运行的容器
docker start $(docker ps -a -q) 启动所有已停止和正在运行的容器
docker rm -vf $(docker ps -a -q) 要删除所有容器，包括其卷，请使用
docker rmi -f $(docker images -a -q) 删除所有图像
docker system prune 删除所有悬空和未使用的图像、容器、缓存和卷
docker system prune -a 删除所有使用和未使用的图像
docker system prune --volumes 删除所有 docker 卷

参数

命令 描述
docker run IMAGE 启动一个新容器
docker run --name CONTAINER IMAGE 启动一个新容器并设置名称
docker run -p HOSTPORT:CONTAINERPORT IMAGE 使用映射端口启动一个新容器
docker run -P IMAGE 启动一个新容器并映射所有端口

检查

命令 描述
docker ps 列出正在运行的容器
docker ps -a 列出所有正在运行的容器
docker logs CONTAINER 显示容器输出
docker logs -f CONTAINER 跟随容器输出
docker top CONTAINER 列出容器中运行的进程
docker diff 显示与图像的差异（修改后的文件）
docker inspect 显示容器信息（json 格式）

命令

命令 描述
docker attach CONTAINER 附加到容器
docker cp CONTAINER:PATH HOSTPATH 从容器中复制文件
docker cp HOSTPATH CONTAINER:PATH 将文件复制到容器中
docker export CONTAINER 导出容器的内容（tar 存档）
docker exec CONTAINER 在容器内运行命令
docker exec -it CONTAINER /bin/bash 在容器内打开交互式外壳
docker wait CONTAINER 等到容器终止并返回退出代码

镜像

命令 描述
docker images 列出所有本地镜像
docker history IMAGE 显示镜像历史
docker inspect IMAGE 显示信息（json格式）
docker tag IMAGE TAG 标记镜像
docker commit CONTAINER IMAGE 创建镜像（从容器）
docker import URL 创建镜像（来自 tarball）
docker rmi IMAGE 删除镜像
docker pull REPO:[TAG] 从注册表中提取镜像/存储库
docker push REPO:[TAG] 推送和镜像/存储库到注册表
docker search TEXT 在官方注册表中搜索镜像
docker login 登录到注册表
docker logout 从注册表注销
docker save REPO:[TAG] 将镜像/存储库导出为 tarball
docker load 从 tarball 加载镜像

卷

命令 描述
docker volumes 列出所有卷
docker volume create VOLUME 创建卷
docker volume inspect VOLUME 显示信息（json格式）
docker volume rm VOLUME 销毁卷

备份一个容器

从容器卷中备份 docker 数据并将其打包进一个 tar文件。 docker run --rm --volumes-from CONTAINER -v $(pwd):/backup busybox tar cvfz /backup/backup.tar CONTAINERPATH

从备份中恢复容器

使用 tar文件恢复卷。 docker run --rm --volumes-from CONTAINER -v $(pwd):/backup busybox bash -c "cd CONTAINERPATH && tar xvf /backup/backup.tar --strip 1"
