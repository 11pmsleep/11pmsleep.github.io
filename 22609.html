网站SEO优化

hexo博客SEO优化

SEO

SEO全称Search Engine Optimization，关乎网站在搜索引擎的排名。排名越高，流量越多。

搜索引擎爬虫

常见爬虫

  • BaiduSpider（百度蜘蛛）
  • Googlebot
  • 360Spider
  • Sogou web spider
  • Bingbot
  • Sosospider

……

网站关键词排名过程

  1. 放出蜘蛛（代码）
  2. 蜘蛛来访抓取（爬虫规则）
  3. 页面的收录
  4. 算法计算得分
  5. 排名

注意前面步骤如果卡死了，那么后面的步骤基本上不会进行

爬虫规则

  • 链接长度，域名分为中文域名和英文域名，爬虫更喜欢英文域名（中文域名识别难度巨大）

    爬虫如果不喜欢的话，爬虫数量会骤降

  • 爬虫模拟真实用户的访问体验：

  • 访问速度
  • 识别难度：文字最好识别，图片稍难，不识别视频
  • 有效收录（有价值的页面）
  • 爬虫来访数量越多，越容易被收录（第一只识别不了，第二只就可能识别收录，所以越多越好）

怎么让爬虫来的越多呢？

  • 要让这个页面的链接越多。一个页面至少二三十个链接才是一个合格的页面
  • 故意让蜘蛛多爬几次该网站

页面的收录

如何快速提升网站收录

网站的各个网页只有被百度此类搜索引擎收录之后才能计算到底合不合格。但是并不是意味着网页越多，就能碰到一个被收录，并不是像买彩票那样，买的越多越可能中将。

其中的收录分为有效收录和无效收录，无效收录过多的网站可能会被惩罚，降低收录次数

如何查看自己的页面是否被收录？以及收录情况

  • 直接搜索引擎搜索页面的url
  • 工具
  • 站长之家

一个例子，只收录了三个页面，但是每天流量有2559

另一个反例：做个这个网站被收录了五万多个，但是流量却少得可怜

大幅增加页面收录几率

工具：燃灯的工具

如何高质量收录？

  • 追求质量而非概率

  • 意思就是收录率越高越好（收录的数量/页面数量）:

    爬虫在返回的过程中就会对该网站初步判断是否收录，如果总是大概率的被收录，比如推送10个被收录9个，那么用户肯定是很喜欢的，那么新的文章大概率也会被收录。

  • 需要注意到，这里的收录数量越高指的是“有效收录”，无效收录的页面再多也是没用的

算法计算得分

权重

权重越高越容易被收录，收录速度越快，来访用户越多

提升权重：

  • 提升网站流量即访客数量，权重就会上来，注意，是需要有效访问
  • 追求高指数关键词排名

核心数据计算

  • 展现量
  • 点击量

点击率=点击次数/展现量

  • 访问量—涉及一个叫跳出率的东西
  • 停留时间

闪电算法：秒排的数据要求

闪电算法是百度针对网站打开速度提出的一项优化规则

  • 大于3秒，难以获得排名
  • 2-3秒，中规中矩
  • 小于2秒，加分
  • 秒排的网站必须是小于一秒的

影响秒排的相关文件因素

  • 模板 css和js少的
  • 图片太多影响识别和加载速度

排名

关键词选择的知识

关键词选择优化朝着面向用户的角度去优化，如果优化错误，就可能会有排名但没流量，或者直接根本没有排名，网站排名上不去

  • 关注关键词的“指数”

以百度指数为例：百度指数 (baidu.com)

可以看到这个关键词每天访问量多少

新站可以选几个500左右的关键词

  • 相关结果

搜索引擎搜索一个关键词都会给出很多个相关结果，有成千上万的竞争对手，，，想在上千万的相关结果中脱颖而出很难

新站可以在一千万排名左右

注意

  • 搜索引擎爬虫是模拟真实用户的访问。模拟用户体验越好，评分越高。影响因素有很多，比如说网站的访问速度，网站域名是否是英文域名，网站文字多不多

  • vip以及一些需要登录的页面爬虫是无法爬取的

  • 来访爬虫越多越容易被收录

    可以在访问日志中筛选查看百度爬虫访问数量

    搜索”查看百度爬虫访问日志“学习具体怎么样看

  • 不仅仅评价网站整体的评分，单页面得分也可以得分很高

一些注意点

  • Github pages不能被百度蜘蛛收录。托管在 GitHub 之上的博客无法被百度检索到。GitHub 不能被百度收录，但 Bing, Google 等搜索引擎，均可正常收录。解决方式：使用 Coding Pages 托管
  • 最好先学seo再建站，不然后期优化比较麻烦，给爬虫的第一印象最重要

优化

网站TDK

Title—Description—Keywords

  • 查看源代码即可看到TDK
  • 网站如果没用关键词，很有可能没有排名
  • 网站打分百分之四十都来自tdk

TDK书写规范

  • 标题：

关键词1-关键词2-关键词3-品牌词

  • 描述
  • 60-80字
  • 吸引用户
  • 抓住用户需求，包含关键词

可以搜关键词，可以看到大部分的需求：

  • 关键词

一些优化技巧

  • 锚文本：即同站内的超链接，可以跳转到其他文章，增加点击率和用户停留时间，同时增加蜘蛛的爬行次数和存活率
  1. 文字于链接内容匹配
  2. 文字最好位要排名关键词
  3. 锚文本的设置密度不能超过1%
  4. 不能多个链接指向一篇文章，否则会形成回链
  5. 站在用户需求（百度监测）
  • 文章写作技巧
  • 图文结合1-3张
  • 字数要求800
  • 不能乱写

喜欢看-搜索到
关键词-挖掘

  • 有效收录
  • 外链

别人网站上放自己的链接，提升访问量

高质量外链效果远远超过多条低质量外链

举个例子：去知乎回答问题引流到该网站

  • 友链
  • 30个左右，用行业
  • 权重高，优质站点
  • 域名选择：
  • 贴近关键词或者品牌
  • 排名难易程度：主域名（一级域名）>二级域名>栏目>内页
  • 链接越长越难排名
  • <head>标签，分为一到六级标题，一篇文章建议只出现一个h1（一级标题），爬虫的监测关键词也跟h标签有关
  • <alt>标签，描述图片，有助于对网站图片被识别，描述要是对图片的概述，越清晰越好
